
**Это эксперимент.**

## Основные функции:
* **Умный поиск:** Автоматический подбор калорий и белков из `food_db.json`.
* **Аналитика:** Интерактивные графики съеденного за день и изменения веса (Chart.js).
* **Mobile First:** Адаптивный дизайн, оптимизированный под управление одной рукой на смартфоне.
* **Zero-Backend:** Работает полностью в браузере, сохраняя данные в `localStorage`.

## Как запустить локально:
1. Клонируйте репозиторий.
2. Откройте `index.html` через **Live Server** в VS Code (важно для корректной работы `fetch` с JSON).

## Стек технологий:
* HTML5, CSS3 (Custom Properties, Flexbox, Grid)
* JavaScript (ES6+)
* Chart.js, SweetAlert2
